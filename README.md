📋 Sistema de gestión de tareas - Consola C#

Programa que permite al usuario administrar su asignación de tareas de manera fácil, automática y estar al tanto de ellas, utilizando el método CRUD.

🚀 Características

- ✅ Registro de contraseña libre, es permitido el uso de letras o signos.
- ✅ Validación de PIN ilimitada hasta ingresar la contraseña correcta.
- ✅ Menú con múltiple opción.
- ✅ Añadir tarea con su nombre, descripción y fecha límite para ser completada.
- ✅ Filtros en estadísticas.
- ✅ Experiencia de usuario mejorada.

🛠️ Tecnologías
Lenguaje: C# Plataforma: .NET (cualquier versión que soporte consola) Compilador: dotnet, Visual Studio o csc 📦 Requisitos previos
.NET SDK (recomendado 6.0 o superior) Visual Studio Community O Visual Studio Code con extensión C# 

⁉️ ¿Cómo encontrar la carpeta donde se aloja el código?
En carpeta llamada "Sistema_Inventario_Versión2.0" deslizas hacia abajo. El archivo llamado "Program.cs" das doble click y aparece el código.


▶️ Cómo ejecutar

Opción 1: Con Visual Studio (más sencillo)
Crea un nuevo proyecto de Aplicación de consola (.NET). Copia todo el código en el archivo Program.cs. Compila y ejecuta con F5 o el botón Iniciar (▶️). Opción 2: Con Visual Studio Code / terminal

Clona el repositorio (o descarga el archivo .cs): git clone https://github.com/JuTevan25/Sistema_Inventario.git
